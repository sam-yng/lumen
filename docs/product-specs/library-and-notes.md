# Library & notes

The student's nested library: folders, documents (TipTap notes), files, and
tags. Built across M2 (library + tagging) and M3 (editor).

## v1 behaviour

- **Folder tree:** create / rename / delete / move; arbitrary nesting; sidebar.
- **Documents:** create / edit / delete TipTap notes with debounced autosave;
  place in folders; a derived plain-text field feeds search.
- **Files:** upload into a folder; audio uploads create a `recording` and
  enqueue transcription (M4).
- **Tags:** assign / remove on documents, files, and recordings; filter by tag.
