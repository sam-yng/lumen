Supabase reference (stub)

Canonical docs: https://supabase.com/docs

Used in Lumen for: Postgres, Auth (email/password), Storage, Row-Level Security,
and the local CLI dev stack (Docker). Schema lives in supabase/migrations/;
types are generated via `bun run db:types`.

Local stack: `bunx supabase start` / `bunx supabase status`.
