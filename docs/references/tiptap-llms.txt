Lumen usage:

- `DocumentEditor` is a client component mounted inside the library workspace.
- Use `useEditor({ immediatelyRender: false })` for Next.js hydration safety.
- Store editor JSON in `documents.content_json`.
- Derive `documents.content_text` server-side from submitted JSON in
  `src/server/services/editor-content.ts`; client code does not send the
  search-facing plain-text field.
- Autosave writes through `PATCH /api/library/documents/:id` with `contentJson`
  and invalidates the `["library"]` TanStack Query key after success.
